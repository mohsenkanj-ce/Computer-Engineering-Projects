# Simple ADD/SUB Processor

## Overview

This project implements a simplified MIPS-style processor using VHDL.

The processor supports ADD and SUB instructions and includes a register file, control unit, multiplexers, decoder, and arithmetic logic unit (ALU).

## Tools Used

- VHDL
- Quartus Prime Lite Edition
- ModelSim

## Components

- Register File
- Control Unit
- ALU
- Decoder
- Multiplexers
- Testbench

## Features

- 32-bit instruction format
- ADD operation
- SUB operation
- Register write control
- Functional simulation and verification
